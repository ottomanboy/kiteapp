## ğŸŒ APIs Used

This app uses the following **free, public APIs** (no API keys required):

1. **NOAA Weather.gov API** - For weather and wind data
   - Current observations
   - Hourly and daily forecasts
   - No API key required

2. **OpenStreetMap Nominatim API** - For location geocoding
   - Converts location names to coordinates
   - No API key required
   - User-Agent header required

3. **NOAA Tides and Currents API** - For tide information
   - Tide predictions for coastal areas
   - No API key required
